## repository 복제
```shell

git clone {url}


```

## status 확인
```shell

git status


```

## 받아오기
```shell

git pull


```

## branch 생성
```shell

git branch {to} {from}


```
### example
```shell

git branch dev main


```

## 파일 추가
```shell

git add {file or directory}


```

### example
```shell

git add .(all)


```

## commit
```shell

git commit -m "message"


```

## push
```shell

git push origin dev


```
